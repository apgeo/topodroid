Cavway Memory

The Cavway X1 memory dialog allows to retrieve the latest data from the device memory. You must specifies the number of data to download, and press the button "Dump". The data are shown in the dialog.

For survey data date, distance, azimuth, inclination, roll and the two dip values are shown. Leg data have a 'L' in front. If the shot has an error there is an 'E'. Calibration data have a 'C', the date, and the raw sensor readings: G1, M1, G2, and M2.

It you specify the name for a dumpfile, the data are also stored to the file.
